## 📊 Featured Projects

### 1. 🏙️ NYC Funding Strategic Audit
> End-to-end audit of **$4.99B** in NYC discretionary spending across **97K transactions**

- Uncovered that **87.52% ($4.09B)** was hidden under a generic "General Program" label
- Built full pipeline: **Python → MySQL → Power BI**
- Delivered governance recommendations for fiscal transparency

**Stack:** Python · Pandas · MySQL · Power BI · Star Schema · DAX

🔗 [View Repository](https://github.com/bilaljab/NYC-Funding-Strategic-Audit)

---

### 2. 🎮 Game Market Intelligence Pipeline
> Production ETL pipeline processing **60K+ records** on video game market trends

- Proved that Critic Scores **≥8.5** are the primary driver of long-term revenue
- Identified "Brand Power" outliers where community loyalty drove outsized sales
- Built an interactive 2-page Power BI dashboard with a neon-dark theme

**Stack:** Python · SQLAlchemy · MySQL · Power BI · DAX · Pandas

🔗 [View Repository](https://github.com/bilaljab/Game-Market-Intelligence-Pipeline)

---

### 3. 📉 Layoffs EDA — Advanced SQL
> Deep exploratory analysis on **2,300+ global layoff records** (2020–2023)

- Applied CTEs, `DENSE_RANK()`, and rolling window functions
- Surfaced cumulative trends and yearly top offenders per sector
- Consumer & Retail identified as the most severely impacted industries

**Stack:** MySQL · CTEs · Window Functions · DENSE_RANK · Aggregations

🔗 [View Repository](https://github.com/bilaljab/EDA-on-Layoffs-Data-using-MySQL)

---

### 4. 🧹 Layoffs Data Cleaning — MySQL
> Professional data cleaning pipeline on a real-world layoffs dataset

- Eliminated duplicates using `ROW_NUMBER()` with staging tables
- Standardized text with `TRIM`, fixed NULLs, converted date formats
- Produced a fully analysis-ready dataset for downstream EDA

**Stack:** MySQL · ROW_NUMBER() · Staging Tables · STR_TO_DATE · Data QA

🔗 [View Repository](https://github.com/bilaljab/Layoffs-Data-Cleaning-using-MySQL)

---

### 5. 📋 Survey Analyst — Power BI
> Transformed **630 survey responses** from data professionals into an executive dashboard

- Isolated salary benchmarks by job title — Data Scientist leads at $94.38K avg
- Python identified as the dominant language with 420 out of 630 votes
- Salary satisfaction scored 4.27/10 — a critical workforce insight

**Stack:** Power BI · DAX · Power Query · Data Modeling · KPI Cards

🔗 [View Repository](https://github.com/bilaljab/Survey-Analyst-PowerBI)
